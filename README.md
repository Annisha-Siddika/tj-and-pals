## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/KabirShuvo/this-is-app.git
   ```
2. Navigate to the project directory:
   ```sh
   cd this-is-app
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Run the application:
   ```sh
   npm run dev
   ```
